## Installation
```
npm install
```

### install packages
```
composer install
```

### copy .env.example
```
cp .env.example .env
```

### generate key
```
php artisan key:generate
```

### Compiles
```
npm run dev
```

### create tables in databse
```
php artisan migrate
```

### create server
```
php artisan serve
```

## Valet Commands (Create local url)
```
valet install
```
```
valet link
```
```
valet park
```
